# Business Blog
### **Getting started**
### Technologies Used
#### HTML5, CSS3, Bootstrap 3.3, jQuery, Font-awesome, Google Map
 - Run :  `git clone https://github.com/getwebem/businessBlog.git`
 - Run :  `cd businessBlog`
 - Run :  `Brackets .`
 - Open:  `Index.html`
 - Browser:  `Live Preview`
